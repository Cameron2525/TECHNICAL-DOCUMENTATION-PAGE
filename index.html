Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of web pages using markup. Below is a simple HTML structure:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document Title</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Additional HTML tags:

<div> </div>
<span> </span>
<a href="..."> </a>
  • Basic HTML structure
  • Common HTML tags
  • HTML attributes

HTML allows for the creation of web structures that are easy to style and script. Understanding HTML is essential for web development.

Basics of CSS

CSS (Cascading Style Sheets) controls the look and feel of a website. It allows you to apply styles to your HTML elements. Here’s an example of basic CSS:

body {
    font-family: Arial, sans-serif;
    color: #333;
}

h1 {
    color: #0056b3;
}

p {
    font-size: 16px;
}

Additional CSS properties:

background-color: #fff;
border: 1px solid #ddd;
padding: 10px;
  • Font properties
  • Color and background
  • Box model

CSS helps you create visually appealing websites. Learning CSS will allow you to enhance the presentation of your web pages.

Introduction to JavaScript

JavaScript is a programming language that adds interactivity to web pages. Here’s a basic example:

<script>
function greet() {
    alert('Hello, welcome to my website!');
}
window.onload = greet;
</script>

Additional JavaScript functions:

function changeColor() {
    document.body.style.backgroundColor = 'lightblue';
}
  • Event handling
  • DOM manipulation
  • Data handling

JavaScript is powerful for creating dynamic web applications. Mastering it will allow you to add complex features to your sites.
